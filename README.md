# Convert-Image-Into-ASCII-Art-In-Python

## Installation

Clone the repository:
```bash
git clone https://github.com/Michael-200/Convert-Image-Into-ASCII-Art-In-Python.git
cd Convert-Image-Into-ASCII-Art-In-Python
```
## Install dependencies
```bash
pip install pillow
```
## Usage  
Place the image you want to convert in the project directory and update the image path in the script:  
img1 = "download.jfif"  
image_to_ascii(img1, width=100)  

## Run the script  
```bash
python script.py
```
